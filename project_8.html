MatrixCrawler

Мини библиотека

Описание проекта

Это тестовое задание. Библиотека реализует асинхронную функцию get_matrix, которая принимает по сети матрицу в виде текста, преобразует её в list[list[int]] и возвращает результат обхода матрицы по спирали, против часовой стрелки, в виде list[int]

C чем работал:
  • Освоил основы асинхронного программирования
  • Разобрался с генераторной природой библитки asyncio
  • Познакомился с библиотекой aiohttp
  • Отработал применение блока try/except для обработки ошибок
  • Отработал применение манкипатчинга в тестировании функции работаюющей с сетью

Использованные технологии
