Gwent Website

Простой магазин настольных игр

Описание проекта

Проект представляет собой простой магазин для однотипного товара на базе Python фреймворка - Django. С возможностью быстрого и простого добавлеия, удаления или редактирования продукта.

В данном проекте получил такие навыки как
  • Организация внешнего медиа хранилища
  • Подключение платёжной системы и работа с web hoock
  • Работа с Ngrok
  • Работа с API сервиса валидации даных пользователей - Dadata
  • Использование адаптивной верстки страницы
  • Настройка VDS сервера Linux/Ubuntu c помощью Ansble
  • Настройка CI с использованием GitHub actions + Ansible
  • Настройка связки Nginx + Gunicorn c использованием своего домена и SSL сертификита
  • Создание SVG логотипа

Использованные технологии

HTML5 CSS3 JavaScript Bootstrap PostgreSQL Django Ansible Gunicorn GitHub Actions Nginx SVG
