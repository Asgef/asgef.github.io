Веб-приложение для управления задачами команде

Описание проекта

Менеджер задач — небольшое веб-приложение на базе фреймворка Python Django, предназначенное для управления задачами в небольшой команде.

Реализованы функции:

  • Регистрация и аутентификация пользователей
  • Создание, редактирование и удаление задач
  • Назначение задач пользователям
  • Назначение статусов и меток задачам
  • Фильтрация задач по одному или нескольким критериям
В данном проекте получил такие навыки как
  • Взаимодействие с базой данных по средствам и Django ORM.
  • Вёрстка контента по средствам вэб-фреймворка Bootstrap 5 DjangoTemplates
  • Построение логики вэб-приложения в ООП стиле
  • Управление языковой локализацией I18N
  • Использование сервиса мониторинга ошибок Rollbar.

Использованные технологии

HTML5 CSS3 Bootstrap PostgreSQL Django Gunicorn GitHub Actions
